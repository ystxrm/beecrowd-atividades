# beecrowd-atividades

O objetivo é registrar minha evolução na programação, praticar lógica e compartilhar códigos simples, limpos e bem estruturados.
✔️Praticar lógica de programação
✔️ Melhorar habilidades em Python
✔️ Evoluir nos desafios do Beecrowd
✔️ Manter um histórico claro da minha progressão
